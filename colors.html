Basic Types¶

The most basic library for working with colors in Julia is ColorTypes.jl. More information about it can be found at https://github.com/JuliaGraphics/ColorTypes.jl. The most basic type it defines is the abstract type Colorant which allows definition of concrete types of colors like RGB{T}. The T parameter allows us to make a color out of different Types such as Floats or Ints, demonstrated below in boxes 2 and 3. Other color types include HSV{T} (Hue, Saturation, Value), RGBA{T} (Red, Green, Blue, Alpha), LCHuv{T} (Luminance, Chroma, Hue), and Gray{T}. There are many more colortypes available based on CIE colorspaces if you need them.

Of course, we want to be able to easily use these color types, which requires using the Colors.jl library. This provides conversions between different colortypes, parsing of color names, and colormaps. More details are available at https://github.com/JuliaGraphics/Colors.jl. What is especially cool about using this library with iJulia is that colortypes are automatically displayed as a color swatch.

ColorMaps¶

colormap() creates an Array{ColorTypes} with specified N=number of colors and mid=midpoint

colormap(cname::String [, N::Int=100; mid=0.5, logscale=false, kvs...])

These maps use a particular hue and generate a path through LCH space (Luminance, Chroma, Hue) that is smooth, without abrupt discontinuities. A primary source for this algorithm is Wijffelaars, M., Vliegen, R., van Wijk, J., van der Linden, E-J. (2008). Generating Color Palettes using Intuitive Parameters. More background can be found in Wijffelaars, M., & Wijk, J. (2008). Synthesis of color palettes (Doctoral dissertation, Master’s thesis. Supervisors Wijk, J. van, and Vliegen).

Sequential palettes¶

Sequential maps use a single hue and are created with the sequential_palette() function. Named sequential palettes are available through colormap(). (i.e. Blues, Greens, Grays, Oranges, Purples, Reds)

Diverging palette¶

Two hues can be combined in a divergent palette. In this case, the first two arguments are the hues at the two ends of the palette. A lot of control can be specified through the w argument (strength of the middle color from 0-1) and wcolor (the middle color). There is a predefined named diverging palette called RdBu (from Red to Blue).

In [25]:
colormap("RdBu")
Out[25]:
In [26]:
diverging_palette(30.0, 240.0, w=0)
Out[26]:
In [27]:
# when we make the middle green, the divergent nature is more apparent
diverging_palette(30.0, 240.0, w=1, wcolor=RGB(0,1,0))
Out[27]:
In [28]:
# this can be softened by changing w
diverging_palette(30.0, 240.0, w=0.5, wcolor=RGB(0,1,0))
Out[28]:
